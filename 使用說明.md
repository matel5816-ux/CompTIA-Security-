# CompTIA Security+ SY0-701 模擬考試系統 - 使用說明

## 🎯 系統特色

### ✨ 答題頁面 - 中文翻譯及題目解析
- 點擊「✨ 中文翻譯及題目解析」按鈕即可查看：
  - 📝 題目的中文翻譯
  - 📋 選項的中文翻譯與正解標示
  - 💡 詳細的觀念解析
- 再次點擊按鈕可隱藏詳解
- **注意**：部分題目翻譯可能尚未補充，系統會顯示「翻譯待補充」

### 📊 結果頁面 - 完整詳細回顧
- **快速導覽卡片**：一眼看所有題目的答題狀況
  - 🟢 綠色 (✓) = 答對
  - 🔴 紅色 (✗) = 答錯
  - ⚫ 灰色 (●) = 未回答
  - 點擊卡片可快速跳轉到對應的題目卡片

- **詳細回顧卡片**：每題的完整答案分析
  - 點擊「展開完整詳解」按鈕在結果頁面內查看中文翻譯和詳細解析
  - 無需跳轉頁面，在同一頁面內展開和收起
  - 每次只能展開一個題目的詳解

## 📚 答題模式

### 🚀 正式考場模擬
- 隨機抽取 75-90 題
- 90 分鐘倒數計時
- 完全模擬真實考試環境

### ⏱️ 快速測驗
- 固定 10 題
- 無時間壓力
- 適合零碎時間複習

### ⚙️ 自訂題數模式
- 自由決定題數 (1-600 題)
- 無時間限制
- 靈活調整練習量

---

## 💡 功能說明

### 標記題目
- 點擊「📌 標記稍後審閱」複選框
- 用於標記有疑問或想複習的題目
- 交卷時會提醒有多少題被標記

### 題目導覽
- **桌機版**：右側固定導覽面板，一覽所有題目狀態
- **手機版**：點擊浮動按鈕 (☰) 打開題目導覽抽屜
- 可直接點擊題目編號跳轉

### 交卷流程
1. 點擊「下一題」或最後一題時的「交卷結算」按鈕
2. 系統會提醒未作答或標記的題目數量
3. 可選擇繼續作答或強制交卷

---

## 📊 成績分析

交卷後，您將看到：

1. **成績卡片**
   - 最終分數 (滿分 900，及格 750)
   - PASS / FAIL 判定
   - 原始答對題數和正確率

2. **題目快速檢視**
   - 所有題目的答題狀況一覽
   - 點擊卡片快速跳轉到對應題目

3. **詳細回顧**
   - 每題的完整答案分析
   - 正確選項 (綠色) vs 錯誤選項 (紅色)
   - 您的答題記錄一目了然

## 🛠️ 翻譯數據補充

系統已預設部分題目的中文翻譯和詳細解析。若您想補充更多題目的翻譯，可以：

1. 編輯 `index.html` 中的題目數據
2. 在每題後添加 `zh` 字段，格式如下：

```javascript
{
  q: "英文題目",
  options: ["Option A", "Option B", ...],
  answer: [0, 2],  // 正確答案的索引
  zh: {
    q: "中文題目翻譯",
    opts: ["選項A翻譯", "選項B翻譯", ...],
    explain: "詳細的觀念解析和選項分析"
  }
}
```

3. 刷新頁面即可生效
